# Screenshots Directory

This directory should contain screenshots demonstrating the application workflow:

1. **main-interface.png** - Main chat interface with sidebar showing database stats
2. **query-results.png** - Example of agent querying and displaying results
3. **charts-visualization.png** - Visualizations (genre distribution, top directors)
4. **support-ticket.png** - Support ticket creation form
5. **console-logs.png** - Console logs showing agent operations

## How to Add Screenshots

1. Run the application: `streamlit run app.py`
2. Take screenshots of the various features
3. Save them to this directory with the names above
4. The README.md will automatically reference these images

## Note

Screenshots should show:
- The chat interface in action
- Function calling in the console logs
- Different query types (simple and complex)
- The support ticket system
- Visual charts and stats
